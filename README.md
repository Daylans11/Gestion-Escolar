# Gestion Escolar
## 📝 Descripción
Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite gestionar los estudiantes, profesores y matriculación de estudiantes en distintas materias y áreas técnicas.  
El sistema permite registrar, actualizar, eliminar y visualizar Estudiantes, profesores, matriculas, Materias y areas tecnicas, así como generar reportes mediante ReportViewer (RDLC).

## Tecnologías Utilizadas

- **Lenguaje(s):**
  - C#
  - SQL

- **Herramientas / Framework:**
  - Windows Forms
  - Biblioteca de clases
  - Microsoft ReportViewer (RDLC)

- **Base de datos:**
  - SQL Server

- **Entorno o IDE:**
  - Visual Studio

## Uso o Ejecución

1. Clonar o descargar el repositorio.
2. Abrir la solución en Visual Studio.
3. Configurar la cadena de conexión a la base de datos SQL Server.
4. Ejecutar el proyecto (F5).
5. Seleccionar un estudiante y materias para registrar la matrícula.
6. Usar el botón **Ver Reporte** para visualizar el reporte de matricula por estudiante.
